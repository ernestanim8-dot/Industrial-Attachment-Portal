## Student Guide

### Dashboard Overview

Your dashboard displays:

- **Attachment Progress**: Overall completion percentage
- **Reports Submitted**: Total number of submissions
- **Graded Reports**: Reports with supervisor assessments
- **Average Grade**: Your overall performance score

### Submitting a Report

1. Click the **"Upload Report"** button
2. Fill in the form:
   - **Report Title**: e.g., "Week 1 Progress Report"
   - **Week/Month Number**: Optional numeric identifier
   - **Description**: Summary of activities and progress
   - **File Upload**: Select your PDF or document file
3. Click **"Submit Report"**
4. You'll receive a confirmation notification

### Viewing Reports

**All Reports Tab:**

- View all your submitted reports
- See submission dates and status

**Pending Tab:**

- Reports awaiting supervisor review

**Graded Tab:**

- Reports with grades and feedback
- View your scores and supervisor comments

### Progress Timeline

- Visual timeline of your attachment journey
- Track milestones and submissions
- See grades and feedback chronologically

### Understanding Status Badges

- 🟠 **Pending**: Waiting for supervisor review
- 🔵 **Reviewed**: Feedback provided, awaiting grade
- 🟢 **Graded**: Fully assessed with score

## Supervisor Guide

### Dashboard Overview

Your dashboard shows:

- **Assigned Students**: Number of students under supervision
- **Pending Reports**: Reports awaiting your review
- **Reviewed/Graded**: Your assessment activity

### Assigned Students Section

- View all students assigned to you
- See their submission counts and progress
- Access individual student reports

### Reviewing Reports

**Quick Feedback (No Grade):**

1. Find the pending report
2. Click **"Quick Feedback"**
3. Enter your comments
4. Click **"Submit Feedback"**
5. Report status changes to "Reviewed"

**Full Grading:**

1. Click **"Grade Report"** on any submission
2. Enter scores for each criterion:
   - **Attendance** (0-100)
   - **Performance** (0-100)
   - **Report Quality** (0-100)
   - **Professionalism** (0-100)
3. The system calculates the average automatically
4. Provide written feedback
5. Click **"Submit Grade"**

### Assessment Criteria

| Criterion | Description |
|-----------|-------------|
| **Attendance** | Regularity and punctuality |
| **Performance** | Quality of work and output |
| **Report Quality** | Documentation and presentation |
| **Professionalism** | Workplace behavior and ethics |

### Managing Reports

**Filter by Status:**

- **Pending Tab**: Unreviewed submissions
- **Reviewed Tab**: Feedback given, not graded
- **Graded Tab**: Fully assessed reports

**Update Grades:**

- Click "Update Grade" on graded reports to revise assessments

## Administrator Guide

### Dashboard Overview

System-wide metrics:

- **Total Users**: Students and supervisors
- **Total Reports**: All submissions
- **Assigned Students**: Supervision assignments
- **Completion Rate**: Average student progress

### User Management

**Adding New Users:**

1. Click **"Add New User"**
2. Fill in user details:
   - Full Name
   - Email Address
   - User Role (Student/Supervisor)
   - Department
   - Company (for students, optional)
3. Click **"Add User"**
4. User receives account credentials

**Managing Students:**

- View all students in table format
- See department, company, supervisor, and progress
- Assign or reassign supervisors
- Remove users when necessary

**Managing Supervisors:**

- View supervisor list
- See assigned student counts
- Remove supervisors

### Assigning Supervisors

1. Find student in Students tab
2. Click **"Assign"** (or "Reassign")
3. Select supervisor from dropdown
4. Click **"Assign Supervisor"**
5. Student receives notification

### Viewing Analytics

Access comprehensive reports on:

- Student performance trends
- Report submission rates
- Department distribution
- System-wide statistics

## Troubleshooting

### Common Issues

**Q: I can't log in**

- Verify your email and password
- Ensure you're using your university email
- Contact your administrator if issues persist

**Q: File upload fails**

- Check file size (max 10MB)
- Ensure file type is PDF, DOC, or DOCX
- Try a different browser

**Q: I don't see my reports**

- Refresh the page
- Check you're logged in with the correct account
- Verify reports were successfully submitted

**Q: Grades not showing**

- Reports must be graded by supervisor first
- Check the "Graded" tab specifically
- Allow time for supervisor assessment

**Q: Notifications not appearing**

- Check notification center (bell icon)
- Ensure notifications are enabled
- Refresh the dashboard

## Best Practices

### For Students

✅ Submit reports on time (weekly/monthly as required)
✅ Provide detailed descriptions of activities
✅ Use professional language and formatting
✅ Review feedback and implement suggestions
✅ Maintain regular communication with supervisor

### For Supervisors

✅ Review reports within 3-5 business days
✅ Provide constructive and specific feedback
✅ Use all grading criteria for fair assessment
✅ Communicate expectations clearly to students
✅ Monitor student progress regularly

### For Administrators

✅ Assign supervisors based on department/expertise
✅ Monitor system usage and user activity
✅ Address user issues promptly
✅ Review analytics for insights
✅ Maintain accurate user records
